## 업그레이드 분기점

### 🔄 Free → Pro 업그레이드 시점

**이런 불편함을 느끼면 Pro로:**

```
❌ "Perplexity도 써보고 싶은데 Free는 3개만 지원하네..."
❌ "4개 AI 답변을 동시에 비교하고 싶은데 2개만 가능하네"
❌ "프롬프트가 3개를 넘어서 자꾸 지워야 해"
❌ "문제가 생겼는데 지원을 받을 수가 없어"
❌ "더 많은 모델로 테스트해야 하는데..."
```

**전환 유도 메시지 (앱 내):**
```typescript
// 예시 메시지
{
  trigger: 'user_tries_to_add_4th_model',
  message: '💼 Pro 플랜으로 업그레이드하면 11개 모델을 모두 사용할 수 있습니다',
  cta: '월 $4.99로 업그레이드'
}

{
  trigger: 'user_tries_to_save_4th_prompt',
  message: '⚠️ Free 플랜은 프롬프트를 3개까지만 저장할 수 있습니다',
  cta: 'Pro로 업그레이드 (10개 저장)'
}

{
  trigger: 'user_tries_to_show_3_models',
  message: '🔒 동시에 3개 이상 모델을 표시하려면 Pro 플랜이 필요합니다',
  cta: '$4.99/월로 잠금 해제'
}
```

**가격 정당화:**
```
월 $4.99 = 커피 1.5잔 값

대신 받는 것:
• ChatGPT Plus ($20) 대신 11개 모델 접근
• 4배 많은 동시 비교
• 3배 많은 프롬프트 저장
• 전문 이메일 지원

절약: $15.01/월 = $180.12/년
```

```

  → 91.7% 절약 ($55.01/월)

  → 75% 절약 ($15.01/월)

  → 72% 절약 ($13.01/월)
```

**2. 기술 차별화**
```
ModelDock (iframe):
✅ 실제 AI 웹사이트 UI 그대로
✅ 새로운 모델 즉시 지원 (AI 사이트가 추가하면 자동)
✅ 모든 기능 접근 (이미지 생성, GPTs 등)
✅ API 키 불필요 (사용자 자신의 계정)

경쟁사 (API):
❌ 커스텀 UI만
❌ API가 지원하는 기능만
❌ API 키 필요 또는 자체 구독
❌ 새 기능 추가 지연
```

**3. 독특한 가치**
```
"실제 AI 웹사이트를 한 화면에 모아서 비교"

→ 다른 제품은 흉내낼 수 없음
→ 니치하지만 명확한 타겟층
→ 파워유저/개발자/연구자에게 필수 도구
```

```typescript
// Stripe Webhook → Supabase → Extension
const licenseCheck = async (userId: string) => {
  // 1. Supabase에서 구독 상태 확인
  const { data } = await supabase
    .from('subscriptions')
    .select('*')
    .eq('user_id', userId)
    .single();

  // 2. Stripe 구독 상태와 동기화
  const subscription = await stripe.subscriptions.retrieve(
    data.stripe_subscription_id
  );

  // 3. 티어별 기능 제한
  return {
    tier: subscription.items.data[0].price.nickname, // 'free', 'pro', 'master'
    features: getTierFeatures(subscription.status),
    validUntil: subscription.current_period_end
  };
}

const getTierFeatures = (tier: string) => {
  const features = {
    free: {
      maxModels: 3,
      maxSimultaneous: 2,
      maxPrompts: 3,
      customThemes: 0
    },
    pro: {
      maxModels: 11,
      maxSimultaneous: 4,
      maxPrompts: 10,
      customThemes: 0
    },
    master: {
      maxModels: 11,
      maxSimultaneous: Infinity,
      maxPrompts: Infinity,
      customThemes: 3
    }
  };

  return features[tier];
}
```
